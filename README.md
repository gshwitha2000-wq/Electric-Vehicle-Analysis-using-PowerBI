# ⚡ Electric Vehicle Analysis Dashboard

## 📊 Overview

This project presents an **Electric Vehicle (EV) Analysis Dashboard** that provides insights into EV adoption trends, vehicle distribution, and performance metrics. The dashboard is designed to help users explore key patterns in electric vehicle usage across different regions, models, and time periods.

---

## 🚀 Features

* **Total Vehicles Overview**

  * Displays total number of EVs analyzed.

* **Average Electric Range**

  * Shows the average driving range of electric vehicles.

* **Vehicle Type Distribution**

  * BEV (Battery Electric Vehicles)
  * PHEV (Plug-in Hybrid Electric Vehicles)

* **Year-wise Trend Analysis**

  * Visual representation of EV growth over model years.

* **State-wise Distribution**

  * Interactive map showing EV distribution across states.

* **Top Manufacturers**

  * Highlights top 10 EV manufacturers by total vehicles.

* **CAFV Eligibility Analysis**

  * Clean Alternative Fuel Vehicle eligibility breakdown.

* **Top Models**

  * Displays most popular EV models using a tree map.

* **Dynamic Filters**

  * Filter by:

    * City
    * Electric Vehicle Type
    * Electric Utility

## 📈 Key Insights

* EV adoption has significantly increased after 2015.
* Battery Electric Vehicles (BEVs) dominate the market compared to PHEVs.
* Certain states show higher EV adoption due to policy and infrastructure.
* Models like Tesla Model Y and Model 3 lead in popularity.

## ▶️ How to Use

1. Open the dashboard file using Power BI / Tableau.
2. Use filters to explore data by city, vehicle type, and utility.
3. Hover over visuals for detailed insights.
4. Analyze trends and patterns for decision-making.
